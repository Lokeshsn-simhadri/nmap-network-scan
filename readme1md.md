# Local Network Port Scan using Nmap

## Objective
To discover open ports on devices in the local network.

## Tool Used
- Nmap

## Network Scanned
10.0.2.0/24

## Results
Active hosts were discovered with open ports:
- Port 135 (MSRPC)
- Port 445 (SMB)
- Port 53 (DNS)
